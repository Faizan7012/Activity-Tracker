## Run Locally

Clone the project

```bash or zsh
  git clone https://github.com/Faizan7012/Activity-Tracker.git
```

Go to the project directory

```bash or zsh
  cd activity-tracker
```

Install dependencies

```bash or zsh
  npm install
```

Start the server

```bash or zsh
  npm run start
```
